# aws-cli

# AWS CLI install instructions

## Installation steps
- Download the installation file
  `curl "https://awscli.amazonaws.com/awscli-exe-linux-x86_64.zip" -o "awscliv2.zip"`
- Unzip the installer
  `unzip awscliv2.zip`
- Run the install program
  `sudo ./aws/install`
- Confirm the installation
  `aws --version`
